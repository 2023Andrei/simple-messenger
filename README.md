# Simple Messenger

Простой приватный мессенджер с:
- Входом по SMS
- Текстовым чатом
- Аудиовызовами через WebRTC

## Как запустить

### 1. Android-приложение
- Открой в Android Studio
- Добавь `google-services.json` из Firebase
- Собери APK

### 2. Сервер
```bash
cd server-nodejs
npm install
node server.js
